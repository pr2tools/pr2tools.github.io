Play Platform Racing 2 offline

Download Stardox PR2PS server

  • 1Download Basilisk Browser with Flash Player
  • 2Open this page in Basilisk and it will display Platform Racing 2 flash game
  • 3Download the server archive and unzip it
    Release v0.2.0
  • 4Open your unziped directory
  • 5Run PR2PS.Web.exe
    (you need to confirm SmartScreen warning)
  • 6Run PR2PS.GameServer.exe
  • 7Default PR2 username for login: admin
    (password not required)
